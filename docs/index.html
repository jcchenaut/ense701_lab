What is MNNN stack?

MNNN stand for:

  • MongoDB: A cross-platform document-oriented database program
  • Nest.js: A Node.js framework for building backend application
  • Next.js: A React framework for building (frontend) web application
  • Node.js: A cross-platform JavaScript runtime environment


Install Node.js before start

You should have Node.js installed in your computer.



Backend application (server) with Nest.js and Node.js

First, install Nest.js by running the command in your terminal: 


            npm i -g @nestjs/cli
Then, run this command to initialize a new Nest.js project: 

            nest new whatever-project-name-you-want
In here we simply call our project 'worksheet3_backend', so the command becomes: 

            nest new worksheet3_backend
We use npm as our package manager:

Nest will install a set of essential dependencies for you. But we still need to install dependencies for connecting MongoDB: 

            cd backend
            npm i @nestjs/mongoose mongoose
